# How to use
## sys-updater
This is the main command, it will print the help message.
### help
This, when combined with the main command, will print the help message.
### update
This, when combined with the main command, will automatically try to get the OS id and compare it against a list, if your OS is supporrted, it's going to trigger a native update command, otherwise it will print an error message.
